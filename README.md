 ## 二、介绍
### 功能
织梦农场是一款模仿星露谷物语的像素风格游戏
<br>游戏实现了`背包`、`储物箱`、`商店`、`工具（砍树、挖矿等）`、`种植`、`NPC的日程寻路对话`、`家具建造`、`昼夜交替`、`存档系统`等功能
